# The Sovereign Slice – Landing Page

A simple, mobile-first landing page built with Bootstrap 5 for a local takeaway business in Kilsyth.

This project is designed to be fast, easy to edit, and deployable via GitHub Pages.

---

## 🍕 Overview

This is a single-page website for **The Sovereign Slice**, a takeaway van serving:

* Stone baked sourdough pizza
* Loaded chips
* Local delivery

The goal of the page is to give customers exactly what they need quickly:

* What’s on offer
* Where to find it
* When it’s open
* How to order

---

## 🌐 Deploy with GitHub Pages

1. Upload your files to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select:

   * Branch: `main`
   * Folder: `/ (root)`
4. Click **Save**

Your site will go live in a few minutes.

## 🔧 Quick Fixes (Common Issues)

**Site not showing on GitHub Pages?**

* Make sure file is named `index.html`
* Check Pages is enabled in settings

**Changes not updating?**

* Hard refresh browser (Ctrl + F5 / Cmd + Shift + R)

**Images not loading?**

* Check file paths or use full image URLs
